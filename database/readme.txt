To run AceTraining's database using XAMPP you need to:

1. Go to localhost/phpmyadmin/ in your browser
2. Create new database called 'acetraining'
3. Inside your created database press 'Import' on top of your phpmyadmin page
4. Press Choose File in 'File to import' area and select file students.sql from this folder
5. Press 'Go' on the bottom of phpmyadmin page
